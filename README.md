# 📊 Employee Attrition Prediction using Machine Learning

## 🧠 Project Overview

This project focuses on predicting employee attrition using machine learning techniques. The goal is to help HR departments identify employees who are likely to leave the organization and take proactive retention measures.

---

## 🎯 Objective

- Analyze employee data to understand attrition patterns  
- Build machine learning models to predict employee exit risk  
- Identify key factors influencing employee attrition  
- Provide actionable HR insights

---

## 📂 Dataset

- Source: IBM HR Analytics Dataset (Kaggle)
- Records: 1470 employees
- Features: Job role, income, satisfaction, overtime, etc.

---

## ⚙️ Technologies Used

- Python 🐍  
- Pandas & NumPy  
- Scikit-learn  
- Matplotlib & Seaborn  
- Jupyter Notebook  

---

## 🤖 Machine Learning Models Used

- Logistic Regression  
- Random Forest Classifier  
- Gradient Boosting Classifier  

Best performing model was selected based on F1-score and ROC-AUC.

---

## 📊 Key Insights

- Employees working overtime have higher attrition risk  
- Low job satisfaction strongly impacts resignation  
- Sales department has highest turnover rate  
- Early-career employees are more likely to leave  
- Salary is important but not the only factor  

---

## 💡 Business Impact

This model helps HR teams:
- Identify high-risk employees early  
- Improve retention strategies  
- Reduce hiring and training costs  
- Improve workplace satisfaction
